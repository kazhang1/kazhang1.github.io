  • This is a summation tracker of the maximum resource rewards from each and every special event thus far in Dragalia Lost.
  • There are slightly different definitions of "maximum" for each type of event.
    • For every type of event the rewards for story views, first clears, stage endeavors, event endeavors, and event daily endeavors are included.
      • For raid events, this accounts for all of the rewards for reaching all the emblem milestones.
      • For facility events, this accounts for all of the prestige milestones as well as all of the resources in the event currency shop.
      • For story events, this accounts for all of the resources in the event currency shop.
  • This is simply a running tally I decided to start up early in the game just to see how rewards from events have changed, events in DL aren't exactly the type you need a calculator or anything for
  • For the time being I have excluded exclusive event rewards, here is a link to the event archive at gamepedia that has info on exclusive rewards: https://dragalialost.gamepedia.com/Events/Archive
  • Event banner images are pulled directly from either the game itself or the Dragalia Lost english website at https://dragalialost.com/en/
